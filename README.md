## Overview

Ensemble Learning improves predictive performance by combining multiple machine learning models.  
This repository demonstrates the implementation, intuition, and comparison of major ensemble techniques.

Topics covered include:

- Voting Methods
- Bagging
- Random Forest
- Boosting Algorithms
- Practical ML workflows using real datasets

Each module contains:

- Clear implementation
- Well-structured notebooks
- Model evaluation
- Visualization of results
